# Testing and Quality Assurance
Testing and Quality Assurance follows a cyclical model as follows:

* Testing
  * Unit Tests
  * Code Analysis
  * Automated Tests
  * Manual Tests
* Triage
  * Investigate
  * Approve
  * Review
* Resolution
  * Resolve
  * Revert
* Acceptance
  * Validate
  * Regression Test
  * Close
